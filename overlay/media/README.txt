Сюда можно класть свои картинки/гифки и звуки для алертов — код трогать не нужно,
оверлей сам подхватит файл, если он лежит тут под правильным именем. Если файла нет —
просто ничего не покажется/не прозвучит, ошибок не будет.

Картинка/гифка — ИМЯ И РАСШИРЕНИЕ ТОЧНО ".gif" (даже если это статичная картинка,
просто назовите файл с расширением .gif):
  follow.gif       — на новый фоллоу
  subscribe.gif    — на новую подписку
  gift_sub.gif     — на подарочную подписку
  raid.gif         — на рейд
  cheer.gif        — на донат битсами

Звук — ИМЯ И РАСШИРЕНИЕ ТОЧНО ".mp3":
  follow.mp3
  subscribe.mp3
  gift_sub.mp3
  raid.mp3
  cheer.mp3

Если хочется другие форматы (png/webp для картинок, wav/ogg для звука) — скажите,
поправим пути в overlay/app.js под них.
