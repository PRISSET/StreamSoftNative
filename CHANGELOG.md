# Changelog

## 1.0.6

- Количество баллов за одно сообщение в чате теперь настраивается (страница
  «Музыка») — раньше было жёстко зашито 1 балл.
- Новая страница «Гифки»: загружаешь гифку и/или звук (как в «Алерты и
  медиа»), назначаешь цену в баллах — зритель включает её командой
  `!gif <имя>` (список — `!gifs`), на оверлее она проигрывается поверх
  алертов.
- Новый опциональный виджет Faceit (страница «Faceit», источник «StreamSoft
  Faceit» в OBS создаётся автоматически): аватар, ник, актуальное ELO с
  изменением за сегодня, график ELO, винрейт/средний K/D/HS%/серия побед и
  последние 5 матчей CS2 с иконкой карты и счётом K/D. Обновляется само в
  фоне. Нужен свой бесплатный API-ключ с developers.faceit.com.

## 1.0.5

- Исправлена проверка HTTPS-сертификатов: программа полагалась на свой
  зашитый список сертификатов и не доверяла настоящему хранилищу Windows,
  из-за чего на машинах с антивирусом, проверяющим HTTPS (Kaspersky, ESET
  и т.п.), скачивание TTS/RVC и проверка обновлений отваливались с виду как
  «GitHub недоступен».
- Установка модуля TTS/RVC больше не может уронить всё приложение целиком —
  ошибка теперь просто показывается в интерфейсе вместо закрытия софта.

## 1.0.4

- Новая тема оформления «Обычная» (по умолчанию) — без эффекта стекла и
  блюра, лёгкая для слабого железа. Переключение на «Glass» (прежний вид)
  доступно на странице «Настройки».
- В свёрнутом виде боковая панель показывает иконки вместо букв.

## 1.0.3

- Автопостинг о начале стрима в Telegram-канал (вкладка «Соцсети»): указываешь
  ID или юзернейм канала, бот сам публикует пост, как только Twitch
  фиксирует реальный старт эфира.
- Боковая панель сворачивается в узкую полоску по кнопке — с анимацией,
  состояние запоминается между запусками.

## 1.0.2

- Резервное воспроизведение YouTube-треков в `!song`: если эмбед не
  проигрывается (автор запретил встраивание — самая частая причина, что не
  все видео с YouTube включались), core сам резолвит прямую аудио-ссылку
  через встроенный yt-dlp и доигрывает трек напрямую, без скачивания файлов
  на диск.
- Команда `!clip` — любой зритель может вырезать клип последних секунд
  стрима прямо из чата, ссылка приходит в ответ. Обновляет права авторизации
  Twitch (`clips:edit`) — после обновления один раз потребуется повторно
  авторизовать Twitch.

## 1.0.1

- Кнопка «Включить/Выключить» для RVC-сервиса на странице «Голос» — можно
  вручную остановить сервис после стрима (например, чтобы не занимал GPU/CPU
  в играх) и снова поднять его одной кнопкой, без перезапуска всего
  приложения.

## 1.0.0

- Чат-агрегация (Twitch/YouTube), веб-оверлей, авто-подключение к OBS,
  Discord Rich Presence, трей, Check&Install для TTS/RVC-модулей.
- RVC ставится через живую pip-установку прямо на машине пользователя, под
  его CUDA (торч подбирается под реальную версию CUDA-драйвера через NVML).
- Core сам запускает RVC-адаптер — не нужно вручную запускать `start.bat`.
- TTS и RVC оживают сразу после установки модуля — без перезапуска
  приложения.
- Автообновление: приложение само проверяет новые версии на GitHub и тихо
  ставит их в фоне, без ручного скачивания инсталлятора.
- Вкладка «Обновления» — показывает установленную версию и список релизов
  с описанием каждого.
- Опрос в чате: зрители голосуют командами !1/!2/... прямо в чате, результат
  сразу на оверлее (`/poll` источник в OBS). При старте опроса бот сам
  объясняет в чате, как голосовать.
- Музыка по баллам: свои баллы (не Twitch Channel Points) начисляются за
  активность в чате, тратятся командой `!song <ссылка YouTube/SoundCloud>`.
  Плеер — отдельный источник `/nowplaying` в OBS. Стример (владелец канала)
  заказывает музыку бесплатно.
- `!help` — список команд прямо в чате, `!points` — проверить баланс баллов.
  Раз в 15 минут бот ненавязчиво напоминает про баллы и `!song`, если фича
  включена.
- Источники Chat/Alerts/Poll/Now Playing в OBS создаются нативным небольшим
  размером вместо растяжения на весь канвас — можно свободно двигать/
  масштабировать без потери качества.
- Переключатели и слайды в iOS-стиле, акцентный зелёный цвет по всему UI.
- Программа пишет лог (`streamsoft.log` рядом с настройками) — проблемы с
  чатом/RVC/музыкой теперь диагностируются без специальной отладочной сборки.
