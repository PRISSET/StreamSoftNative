# Changelog

## 0.1.1

- RVC теперь ставится через живую pip-установку прямо на машине пользователя,
  под его CUDA (вместо одного фиксированного тяжёлого архива) — торч
  подбирается под реальную версию CUDA-драйвера (определяется через NVML).
- Core сам запускает RVC-адаптер (раньше это нужно было делать вручную через
  `start.bat`).
- TTS и RVC оживают сразу после установки модуля — без перезапуска
  приложения.
- Автообновление: приложение само проверяет новые версии на GitHub и
  ставит их в фоне, без ручного скачивания инсталлятора.
- Иконка приложения перерисована — старые маленькие размеры (16/32px)
  строились простым уменьшением большой картинки и выглядели зазубренно;
  теперь каждый размер рендерится отдельно, края ровные.

## 0.1.0
