# Changelog

## 1.0.1

- Новая вкладка «Обновления» — показывает установленную версию и список
  релизов с описанием каждого (тянется прямо с GitHub Releases).

## 1.0.0

- Чат-агрегация (Twitch/YouTube), веб-оверлей, авто-подключение к OBS,
  Discord Rich Presence, трей, Check&Install для TTS/RVC-модулей.
- RVC ставится через живую pip-установку прямо на машине пользователя, под
  его CUDA (торч подбирается под реальную версию CUDA-драйвера через NVML).
- Core сам запускает RVC-адаптер — не нужно вручную запускать `start.bat`.
- TTS и RVC оживают сразу после установки модуля — без перезапуска
  приложения.
- Автообновление: приложение само проверяет новые версии на GitHub и тихо
  ставит их в фоне, без ручного скачивания инсталлятора.
- Иконка приложения перерисована — маленькие размеры (16/32px) больше не
  выглядят зазубренно.
